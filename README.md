## jimmunity - frontend

개인 블로그 지만, 확장성있는 커뮤니티처럼 만들어 볼 예정

### 지원 예정 기능
* 회원 가입 (소셜로그인 - 구글)
* 게시글 쓰기 (이미지 업로드까지)
* 댓글 쓰기
* 대댓글 쓰기
* 게시글과 댓글 좋아요
* 기타 생각나는 기능 추가 예정

### 사용 예정 기술 스택
* **Backend** : SpringBoot, Spring Data JPA, Querydsl, Spring Security(OAuth2), H2 DB(memory), postgresql 
* **Frontend** : Vue.js(maybe..)
